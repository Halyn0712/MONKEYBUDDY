# MonkeyBody 全流程测试矩阵

## 自动静态检查

- Java 源码对 Android 35 `android.jar` 编译。
- Manifest、布局、drawable、values XML 解析。
- 模型配置 JSON 解析。
- 捕获/VLM 队列代码扫描：不得包含截图文件写入或网络请求。
- 模板测试：高低类别、变量替换、连续模板去重。

## 真机功能用例

| 编号 | 测试 | 预期 |
|---|---|---|
| P01 | 首次拒绝悬浮窗权限 | 不启动服务，可再次申请 |
| P02 | 授予悬浮窗权限 | 猴子显示在第三方应用上层 |
| P03 | 拖动并松手 | 平滑贴边，重启后位置保留 |
| P04 | 拒绝使用情况访问 | 不采集，显示明确引导 |
| P05 | 拒绝屏幕采集 | 无 VirtualDisplay、无崩溃 |
| P06 | 允许采集并打开目标社媒 | 状态点绿色，按周期取帧 |
| P07 | 切换到桌面/非目标应用 | 一秒左右暂停，状态点橙色 |
| P08 | 横竖屏切换 | 采集尺寸更新，无黑帧/OOM |
| P09 | 系统撤销 Projection | 服务释放，状态点灰色 |
| V01 | 连续相同内容 | 第二帧被去重 |
| V02 | 快速变化内容 | 队列不超过 1，旧帧释放 |
| V03 | 非法模型 JSON | 不分类、不反馈、不崩溃 |
| F01 | 注入低价值结果 | 毒舌文案、抖动、警示音 |
| F02 | 注入高价值结果 | 正向文案、弹跳、双段提示音 |
| F03 | 8 秒内连续注入 | 仅首次提醒，识别次数均增加 |
| F04 | 文案展开 | 3 秒自动收起，不抢焦点 |
| D01 | Monkey Face sink | 收到事件及当日快照 |
| D02 | sink 抛异常 | Brain 本地链路继续工作 |
| S01 | 30 分钟连续运行 | 无 ANR/OOM，无截图落盘 |
| C01 | 云端模式默认状态 | 默认关闭，不产生网络请求 |
| C02 | 未勾选上传同意并尝试启用 | 拒绝保存启用状态 |
| C03 | 输入新 API Key | Keystore 加密保存，界面和日志不回显 |
| C04 | SiliconFlow 视觉请求 | 返回严格分类 JSON；确认 Base64 data URL 兼容 |
| C05 | API 返回 401/429/503 | 显示错误，不生成模拟识别结果 |
| C06 | 删除 API Key | 密文清除且云端模式关闭 |
| C07 | 抓包检查 | 仅发送预处理 JPEG、分类提示词和鉴权头 |
